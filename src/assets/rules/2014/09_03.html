Si un patineur subit une blessure suffisamment grave pour que les arbitres arrêtent le Jam, ce patineur doit rester sur le banc d’équipe pendant les 3 Jams suivants.

Si plus d’un Jam est arrêté à cause d’un même patineur, celui-ci doit rester sur le banc d’équipe pour le restant de la période.

Si un patineur saigne, le patineur ne peut pas participer au match tant que le saignement n’est pas arrêté.

Si un patineur quitte le Jam pendant le jeu à cause de problèmes d’équipement, le patineur peut retourner dans le Jam une fois le problème résolu. Si le patineur sort du Jam pour une blessure ou pour toute autre raison, le patineur ne peut pas retourner dans le Jam.

Un patineur qui quitte la piste à cause de problèmes d’équipement doit retourner dans le Pack par l’arrière et est sujet aux pénalités selon la Section 5.13 Procédures illégales.

Les patineurs qui ont été blessés avant le match peuvent jouer s’ils ont reçu une autorisation de leur médecin.

Un patineur ne peut pas porter un appareil, plâtre ou une attelle qui représente un danger pour les autres patineurs (si déterminé par le Head Referee).
